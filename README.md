## Rest APIs example with Express

I am use tehnology:
  - Node.js Express.js
  - Sequelize
  - Docker images: MySQL & Adminer
  - Make file

Before use, you must have Docker and Docker-CE installed!

For starting your need:
  1) $ make init
  2) $ make start

For development project your need:
  1) $ make init
  2) $ make dev
